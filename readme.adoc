= GUS/ApiDB Schema Base

A docker image that contains a Postgres instance pre-loaded with the GUS and
ApiDB schemata as well as the generated GUS model perl objects.

This container includes:

* PostgreSQL 15
* Java 21
* Python 3
* Perl 5
** json
** xml-simple
** date-manip
** text-csv
** tree_dag_node
** dbi
** dbd-pg
** dbd-oracle
* Ant
* Maven
